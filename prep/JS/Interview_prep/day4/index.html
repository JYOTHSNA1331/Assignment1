Explain Local Scope, Block Scope, Functional Scope and Scope Chain in javascript

Local Scope: Variables declared within a JavaScript function, become LOCAL to the function.
Block scope: variables with let and const keyword provide block scope. They are accessible only within that block
Functional Scope: local variables have functional scope.They can be accessed within that function
Scope Chain: trying to access the variable in another scope.

What is difference between null and undefined and where to use what?

In 'undefined', we will define a variable, but we won't assign a value to that variable. On the other hand, in
        'null' we will define a variable and assign the 'null' value to the variable.
         Both are primitive data types.

what is Symbol?

        Symbols are primitive data-type and was introduced in ES6.
        These are defined using factory function Symbol()
        They return unique Identifiers and they are immutable

        let s= Symbol("Hello")
        let p = Symbol("Hello")
        console.log(s==p) //return false

Write code to explain map and filter in arrays

 Map: iterate each and every values of an array and push into another array
          let a=[1,2,3,4,5] 
          let b=a.map(n=>n*2)
          console.log(b)//[2,4,6,8,10]

        Filter: It removes the item from an array based on certain condition
             let a=[1,2,3,4,5] 
             let b=a.filter(n=>n>2)
             console.log(b)//[3,4,5]

Explain passed by value and passed by reference

 Pass by value: primitive data type. we are pass the variable as an argument.A copy of variable is created.
              let a=10;
              let b=a;
              b=b+a 
              console.log(a)//10
              console.log(b)//15
         pass by reference: Non-primitive data type. we pass the address as an argument.
              
                let a =[1,2,3,4,5] 
                let b= a 
                b.push(6)
                console.log(a)//1,2,3,4,5,6
                console.log(b)//1,2,3,4,5,6
